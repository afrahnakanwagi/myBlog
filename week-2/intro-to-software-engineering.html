INTRODUCYION TO SOFTWARE ENGINEERING

In week to we were introduced to the most interesting parts of software web-site development, going deep into its core.
GitHub

GitHub is a platform used for vision control and collaboration in software development between the teams.It was a great experience after installing it and trained how to operate with it,it functions along with GitBash were the programmer operates it by giving it commands like" git init.,git add .,git commit -m,git push,git remove,etc"And this was so interesting and amazing because by this time we could create a website from just scratch and then modify it by Git . There were new amazing terms like"repository, directory, etc"

CSS

CSS"Cascading Style Sheets"is used to add style to websites when creating them from scratch .css is added on the HTML framework to spice up our websites and give them a blend.css is comprised of selectors,elements and declarations.It has types and we at times use links to link html files with css files .Woow!!!!!!! just imagine all that creativity , this was the best experience ever ,within a shrt time i had learnt to create a webpage just from scratch and everything was making sense.

.

Bootstrap

So here is the most amazing part of it ,Bootstrap,and by this ,i mean a reuseable package of already written CSS and HTML and Java script files. And this helps a website developer to create a website quickly.this uses CDN links and local links.So in simple terms this just simplifys website building to us beginners .At first it was a bit challenging to use it but finally practice made it perfect.
Bootstrap Grid

So now this takes us into bootstrap were this grid helps us to present our web pagesby including layouts ,visual designs ,etc .Grids are made by rows and columns.
